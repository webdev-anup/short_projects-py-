# 🐍 Python Mini Projects

A collection of beginner–intermediate Python projects built while practicing programming concepts.  
Each project is stored as a separate `.py` file for easy reference.

---

## 📑 Table of Contents

1. [Expense Tracker](./expense_tracker.py)  
   ➡️ Add, view, and calculate total expenses stored in a text file.

2. [Countdown Timer](./countdown_timer.py)  
   ➡️ Tkinter GUI timer with beep sound and desktop notification.

3. [Calculator](./calculator.py)  
   ➡️ Perform basic arithmetic operations (add, subtract, multiply, divide).

4. [Selection Sort](./selection_sort.py)  
   ➡️ Implementation of selection sort algorithm with step-by-step logic.

5. [Bubble Sort](./bubble_sort.py)  
   ➡️ Classic sorting algorithm for practice.

6. [Other Practice Scripts](./)  
   ➡️ Small experiments and problem-solving exercises.
